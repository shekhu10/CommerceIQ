Additional functionalities

1. Not allowed to enter duplicate id.
2. Not allowed to enter id other than Integer.
3. Sorting allowed on all parameters. => Get Request No. 4

API calls

get requests

1. https://shekhar-node-deploy.herokuapp.com/posts/:id

Gives result of specific ID in JSON ----- {:id} as 1,2,3... (In Integer)

sample - https://shekhar-node-deploy.herokuapp.com/posts/2



2. https://shekhar-node-deploy.herokuapp.com/posts/

gives all posts stored in store.json

sample - https://shekhar-node-deploy.herokuapp.com/posts/



3. https://shekhar-node-deploy.herokuapp.com/posts?title=title1&author=CIQ

Gives all posts with the specific title && specfic author

sample - https://shekhar-node-deploy.herokuapp.com/posts?title=title1&author=CIQ3



4. https://shekhar-node-deploy.herokuapp.com/posts?_sort=[1]&_order=[2]

gives all posts by sorting ----sort on the basis of [1], and order on the basis of [2]
[1] => id, views, reviews
[2] => asc, des (plz match string while testing)

sample - https://shekhar-node-deploy.herokuapp.com/posts?_sort=views&_order=des



5. https://shekhar-node-deploy.herokuapp.com/posts?q=value

gives all posts where by filtering on q
q => id, title, author, views, reviews

sample - https://shekhar-node-deploy.herokuapp.com/posts?title=title1

https://shekhar-node-deploy.herokuapp.com/posts/:id
